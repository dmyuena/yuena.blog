Git的使用

查看版本 git –version

设置标识 git config –global user.name XXXX

​ git config –global user.email XXXX

初始化 git init

把文件从暂存区撤销 git rm –cached

​ git restore

查看版本提交的命名 git reflog

查看完整版的日志 git log

回到任何一个提交的版本 git reset –hard <版本号>

查看本地库状态 git status

添加到暂存区 git add

提交到本地库 git commit -m “备注”

Git分支

查看目前在什么分支 git branch -v

创建分支 git branch <分支名>

切换分支 git checkout <分支名>

分支合并 git merge <需要合并到此分支的分支>

Git操作远程仓库

创建远程库别名 git remote add <仓库别名> <从github上复制的仓库名>

查看远程库 git remote -v

推送本地分支到远程仓库上 git push <仓库别名> <分支>

拉取远程仓库到本地 git pull <远程仓库别名> <需要pull的分支名>

克隆项目 git clone <从github上复制的仓库名>
