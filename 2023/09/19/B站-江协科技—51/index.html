单片机的简单介绍

  • 单片机，Micro Controller Unit，简称 MCU

  • 内部集成了CPU、RAM、ROM、定时器、中断系统、通讯接口等

  • 单片机的任务是信息采集、处理和控制硬件设备

单片机算是一个袖珍版计算机，一个芯片就能构成完整的计算机系统。但在性能上，与计算机相差甚远，但单片机成本低、体积小、结构简单，在生活和工业控制领域大有所用

STC89C52单片机

  • 位数：8位
  • RAM：512字节
  • ROM：8K（Flash）
  • 工作频率：12MHz

STC-ISP软件的使用

烧录程序

  • 单片机型号根据自己的设备进行选择
  • 设备连接电脑后会自动匹配串口号
  • 点击 ‘打开程序文件’ 在文件目录下的Object文件夹下选择.hex文件
  • 点击下载/编程，烧录程序
  • 重启单片机即可

延迟时间（生成delay函数）

  • 选择软件延时计算器
  • 选择系统频率（单片机晶振）
  • 选择相应的8051指令集:STC-Y1,适用于STC89C52系列单

第四章：数码管

原理图

  • 本开发板使用共阴极连接

  • 使用译码器实现位选：选择哪个数码管亮

  • 74LS138译码器

  • 输入端 A B C

  • 使能端 G1 G2! G3!

  • 输出端 Y!

真值表

C B A 输出 Y的脚标
0 0 0 0111 1111 0
0 0 1 1011 1111 1
0 1 0 1101 1111 2
0 1 1 1110 1111 3
1 0 0 1111 0111 4
1 0 1 1111 1011 5
1 1 0 1111 1101 6
1 1 1 1111 1110 7

  • 使用收发器（或锁存器）实现段选：控制数码管怎么亮

  • 74HC245 收发器

  • 使能端 OE!

  • 选择端 DIR 接高电平左边的数据送到右边

    ​ 接低电平右边的数据送到左边

74hc245是八同相三态收发器,数据可双向传送.

74c573是8位三态锁存器,有锁存数据功能.数据单向传送.

两种芯片都有三态,都能起缓冲作用.
